# RickAndMortyApp

Una aplicación de Android que muestra personajes de la serie Rick and Morty utilizando la API pública de Rick and Morty.

## Características

*   Muestra una lista de personajes de Rick and Morty.
*   Paginación para cargar más personajes a medida que el usuario se desplaza.
*   Construida con prácticas modernas de desarrollo de Android.

## Arquitectura y Tecnologías Utilizadas

*   **Lenguaje:** [Kotlin](https://kotlinlang.org/)
*   **UI:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
*   **Red:** [Retrofit](https://square.github.io/retrofit/) para las llamadas a la API.
*   **Inyección de Dependencias:** [Hilt](https://dagger.dev/hilt/)
*   **Paginación:** [Paging 3](https://developer.android.com/topic/libraries/architecture/paging/v3-overview)
*   **Carga de Imágenes:** [Coil](https://coil-kt.github.io/coil/)
*   **Asincronía:** [Kotlin Coroutines](https://kotlinlang.org/docs/coroutines-overview.html)
*   **Arquitectura:** MVVM (Model-View-ViewModel)
